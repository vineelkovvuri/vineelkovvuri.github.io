Features
• Load a PE file (.exe, .dll, .sys, .efi) via drag-and-drop or file picker
• Displays current ImageBase with file offset and field size
• Supports both PE32 (4-byte ImageBase) and PE32+ (8-byte ImageBase) formats
• Enter a new hex address (e.g. 0x10000 or 0x00007FF700000000) to rebase
• Section headers displayed for reference
• Downloads the modified PE file — no data leaves your browser
