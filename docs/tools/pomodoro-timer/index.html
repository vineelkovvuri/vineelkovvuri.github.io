How it works
• Pomodoro — 25-minute focused work session
• Short Break — 5-minute break between sessions
• Long Break — 15-minute break after several sessions

Features
• Circular progress ring with countdown timer
• Type a task name before starting to track what you worked on
• Session dots track up to 20 completed pomodoros
• Completed tasks logged with start/end times
• Bell sound notification when a session ends
• Timer shows in the browser tab title while running
• Tasks and session count persist across page reloads
