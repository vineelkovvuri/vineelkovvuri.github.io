Grid
Grid Style
How it works
• Pick a paper size, drop an image onto the canvas, and overlay a measured grid.
• Print or save the result, then reproduce each cell on your own gridded paper — the classic grid method for scaling up a reference image by hand.

Features
• Paper sizes — A2, A3, A4, A5, US Letter/Legal, or a fully Custom size in millimetres, in portrait or landscape.
• Grid by cell size — set a real-world cell size (e.g. 20 mm) and columns/rows are computed to match the paper.
• Grid by count — set exact Columns and Rows; enable Square cells to keep cells proportional.
• Image fit — Fit keeps the aspect ratio, Cover crops to fill, Stretch forces an exact fill.
• Grid style — adjustable line color, thickness, and opacity, an optional row/column ruler with numbers across the top and down the side, and a page border.
• Export — Save PNG renders at 300 DPI for crisp printing; Print sends the page straight to your printer.
