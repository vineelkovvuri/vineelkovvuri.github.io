🌳 Tree Visualizer

No node selected
Features
• Add Node — creates a free-standing node at the center of the current view
• Add Child — adds a child node connected to the selected node with a smooth curved arrow
• Delete — removes the selected node and all of its descendants
• Realign — re-applies the automatic tree layout to all nodes with a smooth animation
• Export PNG — saves the current canvas view as a PNG image
• Click a node to select it (highlighted in green); drag to reposition it freely
• Tab / Shift+Tab to cycle through nodes level-by-level (BFS order)
• Ctrl+↓ — add a child to the selected node
• ↑ \u2014 move selection to parent  |  ↓ \u2014 move to first child  |  ← / → \u2014 move to previous/next sibling
• Ctrl+← / Ctrl+→ — create a left/right sibling of the selected node
• Double-click a node to edit its label inline; press Enter or click away to confirm, Escape to cancel
• F2 — rename the selected node
• Del — delete the selected node and all its descendants
• Scroll to zoom in/out; drag the background to pan the canvas
