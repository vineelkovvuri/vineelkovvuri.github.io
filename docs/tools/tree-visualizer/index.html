🌳 Tree Visualizer

Ctrl+Enter to draw  |  Scroll to zoom  |  Drag background to pan
How it works
Type your tree using indentation (spaces or tabs) to define parent–child relationships, then click Draw or press Ctrl+Enter: 
Root
    Node 1
        Child A
        Child B
    Node 2
Features
• Any indent unit works — 2 spaces, 4 spaces, tabs, mixed
• Draw / Ctrl+Enter — parse the text and redraw the tree with animation
• Export PNG — exports the full tree at 100% zoom with a tight bounding box
• Click a node to highlight it; drag to reposition manually
• Scroll to zoom; drag the background to pan
• Use Monaco multi-cursor, find&replace, and all editor shortcuts to edit the tree text
