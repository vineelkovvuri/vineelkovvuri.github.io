Beat your best streak!

How it works
• A flag appears — pick the correct country from four choices.
• Each flag has a countdown timer; answer before it runs out or it counts as a miss.
• Correct answers grow your streak 🔥; a wrong answer or timeout resets it.
• A round is 20 flags drawn at random from ~180 countries.
• Flags are derived from each country's two-letter ISO code (e.g. US → 🇺🇸) and drawn as crisp rectangular SVGs via CDN — no image files are stored in this repo.

Tips
• No images are stored in this repo — everything is derived from a two-letter country code.
• Press 1–4 on the keyboard to answer quickly.
