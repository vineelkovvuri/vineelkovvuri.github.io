## 🕹️ Chip8 Emulator

**Chip8 Emulator** is a CHIP-8 virtual machine implementation designed to
emulate classic CHIP-8 programs and games. The project focuses on low-level
concepts such as instruction decoding, memory management, timers, and display
rendering. It serves as a hands-on exercise in emulator development and helps
build a deeper understanding of how simple interpreted systems work.

🔗 [https://github.com/vineelkovvuri/Projects/tree/master/Chip8Emulator](https://github.com/vineelkovvuri/Projects/tree/master/Chip8Emulator)

## 🔍 Extendable Desktop Search

**Extendable Desktop Search** is a desktop search tool designed with
extensibility in mind. The project explores file system traversal, indexing, and
search optimization, while allowing new search features or data sources to be
plugged in easily. It highlights practical problem-solving around performance,
scalability, and clean software architecture.

## 🧩 PeLib

**PeLib** is a library for parsing and working with Windows Portable Executable
(PE) files. It focuses on understanding executable internals such as headers,
sections, and metadata. This project demonstrates strong systems-level knowledge
and is particularly useful for learning about OS loaders, binary formats, and
reverse-engineering fundamentals.

🔗 [https://github.com/vineelkovvuri/Projects/tree/master/PeLib](https://github.com/vineelkovvuri/Projects/tree/master/PeLib)
