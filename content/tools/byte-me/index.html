BYTE ME

A hex value appears — pick the memory size it represents.
Builds from KB up to TB as you level up.

How it works
• A hex value (up to 64-bit) is shown. Pick the memory size it equals.
• Sizes are binary: 1 KB = 1024 B, 1 MB = 1024 KB, and so on.
• Handy anchors: 0x400 = 1 KB, 0x1000 = 4 KB, 0x100000 = 1 MB, 0x40000000 = 1 GB.
• Correct answers score points — longer streaks earn bonuses.
• Every 5 correct, the level rises: bigger units (MB → GB → TB).
• A wrong pick costs a life. 3 lives, then game over.
• After each answer the breakdown is shown to reinforce the conversion.

Controls
• Click / tap an option, or press 1–4.
• Space starts the game.

Goal — build fluency reading hex values as real memory sizes.
